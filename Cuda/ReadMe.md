# CUDA Tutorial

## Setup

- Ubuntu 20.04
- Visual Studio Code IDE and compiler
- Nvidia Driver Version: 510.108.03
- CUDA Version: 11.6
- GPU: NVIDIA T500
- Compiler - GCC 9.4.0, NVCC


## How to compile?
- To compile .cu files, run the following command:
nvcc <filename>.cu
